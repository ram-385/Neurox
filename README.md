# AI NeuroX

AI NeuroX is an end-to-end AI and Machine Learning platform developed using React, FastAPI, Python, Pandas, Scikit-learn, and Plotly. The project is designed to simplify the complete machine learning workflow through an interactive and modern web interface.

Users can upload datasets, perform preprocessing, analyze statistics, generate visualizations, apply feature engineering techniques, and train machine learning models efficiently within a single platform.

---

## Features

- Dataset Upload and Preview
- Data Cleaning and Preprocessing
- Missing Value Handling
- Categorical Encoding
- Outlier Detection and Treatment
- Statistical Analysis
- Interactive Data Visualization
- Feature Engineering
- Machine Learning Model Training
- Model Performance Evaluation
- Comparison Graphs and Metrics
